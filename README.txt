Follow Theme
============

* Es theme administrativo para Drupal 6 usando Twitter Bootstrap y Zen Theme
* Es un subtheme usando Zen Theme, lo puedes descargar desde http://drupal.org/project/zen
* Se instala como cualquier theme en Drupal, y se activa como theme administrativo en /admin/settings/admin
* La intencion no es usarlo para el lado publico de la aplicación, como comente, es un tema administrativo
* Solo se probo usando Google Chrome, aún no se probo en otros navegadores
* Algunas cosas pendientes: aplicar mejor el estilo en formularios, el metodo de arrastre no funciona/se ve mal, no se probo con modulos que no sean los originales de Drupal
* Se quitaron todos los CSS default de Drupal, no carga ninguno, asi se podia depurar con facilidad
* ADVERTENCIA : Es un alpha, aun faltan revisar muchas cosas, esta verde, puede explotar, cuidado!, 
* LICENCIA DE USO : es opensource, uselo como lo necesite.
* Bienvenidos los aportes!
